# Synergy Unity Prototype

## Описание проекта
Прототип 2D-платформера на Unity, разработанный в рамках производственной практики в ООО «Tatwest.

## Реализованный функционал
- Слежение камеры за игроком (Cinemachine)
- Анимация бонусов (машина состояний Animator)
- Система сбора монет с отображением в UI
- Интерактивные препятствия (взрывная бочка, падающие объекты)
- Система здоровья игрока (HP)
- Система частиц для эффектов движения

## Технологии
- Unity 2022.3 LTS
- C#
- Cinemachine
